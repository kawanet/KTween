from()method 
public static function from(target:*, duration:Number, from:Object, ease:Function = null, callback:Function = null):KTJob

Starts a new KTween job specifying the first (beginning) status. The last (ending) status will be back the current status.

Parameters

target:* — The target object to be tweened.
 
duration:Number — The length of the tween in seconds.
 
from:Object — The object which contains the first (beginning) status in each property.
 
ease:Function (default = null) — The easing equation function. Quad.easeOut is default.
 
callback:Function (default = null) — The callback function invoked after the tween completed as onClose.

Returns
KTJob — The KTween job instance.
